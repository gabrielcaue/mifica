## рҹҢҗ Fluxo de Dados вҖ” Exemplo: AutenticaГ§ГЈo + GamificaГ§ГЈo

```
1. CLIENTE
   в””в”Җ POST /api/auth/login {username, password}

2. CONTROLLER
   в””в”Җ AuthController.login()

3. SERVICE
   в”ңв”Җ UsuarioService.buscarPorUsername()
   в”ңв”Җ BCryptPasswordEncoder.matches()
   в””в”Җ JwtTokenProvider.gerarToken()

4. REPOSITORY
   в””в”Җ UserRepository.findByUsername() [BD: SELECT]

5. PERSISTГҠNCIA
   в””в”Җ MySQL: usuario table

6. RESPONSE
   в””в”Җ Cliente recebe {token: JWT, tipo: Bearer}

7. PRГ“XIMAS REQUISIГҮГ•ES
   в”ңв”Җ Header: Authorization: Bearer {token}
   в”ңв”Җ JwtFiltro intercepta
   в”ңв”Җ Valida signature e claims
   в””в”Җ Autoriza acesso ao endpoint

8. EVENTOS ASSГҚNCRONOS (exemplo: desafio concluГӯdo)
   в”ңв”Җ DesafioService.marcarConcluГӯdo()
   в”ңв”Җ Redis.publish("desafio.concluido", {userId, desafioId})
   в”ңв”Җ GamificacaoListener consome evento
   в”ңв”Җ GamificacaoService.calcularPontos()
   в””в”Җ UserService.atualizarReputacao()
```
